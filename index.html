Google Sheets Integration

Follow these steps to connect the website form and this CRM to a Google Sheet:

Step 1 — Create Google Sheet
Open Google Sheets, create a new sheet called Popular Properties Leads. Add these headers in Row 1:
ID | Timestamp | Name | Phone | Service | Budget | Message | Status | Source
Step 2 — Create Apps Script
In your Sheet, go to Extensions → Apps Script. Paste this code and click Deploy → New Deployment → Web App (set access to "Anyone"):
function doPost(e) {
  var sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data.id, data.timestamp, data.name,
    data.phone, data.service, data.budget,
    data.message, data.status, data.source
  ]);
  return ContentService
    .createTextOutput('OK')
    .setMimeType(ContentService.MimeType.TEXT);
}
